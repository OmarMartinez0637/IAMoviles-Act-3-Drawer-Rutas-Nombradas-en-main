# Prompt

Drawer (4 opciones)

Encabezado: Va a haber una imagen de avatar de mi negocio que es un Dairy Queen, Imagen subida desde la red (github) https://raw.githubusercontent.com/OmarMartinez0637/Imagenes-Para-DairyQueen/refs/heads/main/Dairy_Queen_logo.svg.png
Después de la imagen de avatar vas a poner los siguientes datos: Nombre de la empresa (Dairy Queen), Dirección (Blvd. Tomás Fernández 1234), Teléfono ((656) 555-0192), correo (contacto@dq-juarez.com).
	
Listile: Icono, texto y acción para abrir una página en base a la opción del listile,
(Helados, Bebidas, Pasteles, Para llevar).

En cada página va a haber una imagen centrada de 200x200 subida desde la red (github) solo dame el código listo para yo poder poner las imágenes que yo quiero de forma manual.


Vas a usar rutas nombradas para la navegación entre páginas a través del drawer.

Hacerlo en archivos diferentes, me refiero a que vas a poner una página por archivo, aparte del main.

Este proyecto se basa en una heladeria Dairy Queen, entonces vas a usar una paleta de colores azul, rojo y el fondo es de color blanco, todos los textos van a ser de color negro
